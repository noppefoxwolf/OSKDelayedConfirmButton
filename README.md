# OSKDelayedConfirmButton
Button like Android Wear.

![](https://raw.githubusercontent.com/noppefoxwolf/OSKDelayedConfirmButton/master/sample.gif)

# Usage is simple

```
[button setConfirmAction:^{
        NSLog(@"confirm");
}];
[button setCancelAction:^{
        NSLog(@"cancel");
}];
```
